# 内存管理 (memory management)

- 改善CPU的使用率和对用户响应速度

- 每个内存管理算法都需要硬件支持



一个典型的指令执行周期, 首先从内存中读取指令. 接着该指令被解码, 且可能需要从内存中读取操作数. 在指令对操作数执行后, 其结果可能被存回内存中.
